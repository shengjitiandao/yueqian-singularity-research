## 为什么做这个？

我们是谁？我们如何理解自己？我们如何变化？

这些问题不是要被解决的，而是要被看见的。

记录本身就是一种痕迹。当我们把自己的困惑、故事、日常说出来的时候，可能就在累积那些"这是我"的落定瞬间。

1. **自述**：写一篇关于自己的故事，可以是某个重要瞬间、某个困惑、某个改变

3. **日常**：随时发邮件分享，我会整理收录
